# Bitrix24 Tasks Dashboard (встроенная страница)

Минимальное встроенное приложение (iframe) для Битрикс24, которое строит дашборд по задачам:
- кол-во по статусам
- просроченные и %
- SLA/временные метрики (lead time / cycle time — приближенно)
- нагрузка по исполнителям

## Как запустить в Битрикс24 (облако)

Нужно разместить эти файлы на доступном по HTTPS URL (любой хостинг). После этого в Битрикс24:

1) **Приложения → Разработчикам → Добавить приложение** (Local application / Встроенное приложение).
2) Укажите URL на `index.html` (например `https://your-domain.example/dashboard/index.html`).
3) Разрешения: **tasks** (задачи) и (опционально) **user**/**department** если будете расширять справочники.

После установки страница откроется внутри Битрикс24 и автоматически получит контекст авторизации через `BX24`.

## Структура

- `index.html` — страница приложения
- `app.js` — логика загрузки/агрегации/визуализации
- `styles.css` — стили

## Примечания по SLA

Bitrix24 REST для задач не всегда отдает детальную историю смены статусов. Поэтому:
- **Lead time**: \(CLOSED\_DATE - CREATED\_DATE\) (только для закрытых)
- **Cycle time (approx)**: \(CLOSED\_DATE - CHANGED\_DATE\) (приближение, при необходимости заменим на историю/таймлайн)

Если вам нужна точная аналитика «время в статусах», скажите — подберем способ через события/журнал/BI-выгрузку.
